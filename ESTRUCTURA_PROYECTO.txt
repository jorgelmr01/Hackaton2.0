    • Dev server optimizado

🎨 CSS3 Variables
    • Diseño modular
    • Temas dinámicos
    • Animaciones CSS


╔══════════════════════════════════════════════════════════════════════╗
║                  CARACTERÍSTICAS POR PANTALLA                        ║
╚══════════════════════════════════════════════════════════════════════╝

📊 PANTALLA 1: RESUMEN OPENFRIENDS (/openfriends)
   ├── Gráfico circular interactivo (Recharts)
   ├── Click en segmentos para ver detalle
   ├── Balance "Te deben" vs "Debes"
   ├── Lista de grupos con navegación
   ├── Botón flotante Chat OBI
   ├── Link a historial
   └── Animación de entrada

💬 PANTALLA 2: CHAT OBI (/chat-obi)
   ├── Interfaz conversacional
   ├── Burbujas usuario/OBI
   ├── Reconocimiento de keywords
   ├── Tarjetas mini interactivas
   ├── Botones de acción contextual
   ├── Quick actions
   ├── Selección de contactos
   ├── Auto-scroll a último mensaje
   └── Navegación desde chat

🧮 PANTALLA 3: OPENSPLIT (/opensplit)
   ├── Lista de transacciones (mock data)
   ├── Selección múltiple con checkboxes
   ├── 3 pasos (select → configure → confirm)
   ├── Selector de amigos
   ├── División equitativa/proporcional
   ├── Agregar contactos externos
   ├── Cálculo automático por persona
   ├── Preview de envío WhatsApp
   └── Confirmación visual

💰 PANTALLA 4: DETALLE APARTADO (/apartado/:id)
   ├── Header con foto de grupo
   ├── Barra de progreso animada
   ├── Grid de participantes
   ├── Montos individuales
   ├── Botones de acción (Aportar, Proponer, Votar)
   ├── Panel expandible de votaciones
   ├── Sistema de votos visual
   ├── Historial de movimientos
   └── Link a crédito grupal

🏦 PANTALLA 5: CRÉDITO COMPARTIDO (/credito/:id)
   ├── Grupo info badge
   ├── Slider de monto (10K - 200K)
   ├── Gradiente dinámico en slider
   ├── Botones de plazo (3, 6, 12 meses)
   ├── Tasa de interés fija 12.5%
   ├── Cálculo en tiempo real
   ├── Pago mensual grupal
   ├── Pago por persona
   ├── Grid de firmas digitales
   ├── Animación en firma
   ├── Vibración táctil
   ├── Validación completa
   └── Info card con detalles

📜 PANTALLA 6: HISTORIAL (/historial)
   ├── Barra de búsqueda con clear
   ├── Panel de filtros expandible
   ├── Filtros: todos, splits, apartados, créditos
   ├── Lista de transacciones
   ├── Iconografía por tipo
   ├── Badges de estado
   ├── Montos positivos/negativos
   ├── Búsqueda en tiempo real
   └── Estado vacío

➕ PANTALLA 7: CREAR GRUPO (/crear-grupo)
   ├── Preview en tiempo real
   ├── Input nombre de grupo
   ├── Input meta de ahorro (con $ prefix)
   ├── Grid selector de colores (6 colores)
   ├── Lista de participantes
   ├── Chips amigos Openbank
   ├── Form agregar externos
   ├── Validación de campos
   ├── Botón con color dinámico
   └── Confirmación de creación

╔══════════════════════════════════════════════════════════════════════╗
║                     ESTADO Y DATOS                                   ║
╚══════════════════════════════════════════════════════════════════════╝

🔄 useState usado en:
  • selectedSegment (Resumen)
  • messages, inputValue (Chat)
  • selectedTransactions, step, friends (Split)
  • showVotaciones (Apartado)
  • monto, plazo, firmas (Crédito)
  • filterOpen, searchQuery (Historial)
  • nombre, metaAhorro, color (Crear Grupo)

📊 Datos de ejemplo incluidos:
  • Transacciones de tarjeta
  • Grupos con progreso
  • Amigos con initials
  • Historial de movimientos
  • Votaciones pendientes
