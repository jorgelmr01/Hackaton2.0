1️⃣  INSTALAR NODE.JS
   → Ve a: https://nodejs.org/
   → Descarga versión LTS
   → Ejecuta el instalador

2️⃣  HACER DOBLE CLICK EN: START.bat
   → Instalará todo automáticamente
   → Iniciará el servidor
   → Abrirá el navegador

3️⃣  ACTIVAR VISTA MÓVIL
   → Presiona F12 en el navegador
   → Presiona Ctrl+Shift+M
   → Selecciona iPhone o Pixel


┌─────────────────────────────────────────────────────────────────────┐
│  📱 LAS 7 PANTALLAS                                                  │
└─────────────────────────────────────────────────────────────────────┘

1. 📊 RESUMEN OPENFRIENDS
   • Gráfico circular de deudas
   • Balance te deben/debes
   • Lista de grupos
   • Botón flotante OBI

2. 💬 CHAT OBI
   • Asistente conversacional
   • Tarjetas interactivas
   • Acciones rápidas
   • Navegación integrada

3. 🧮 OPENSPLIT
   • Dividir transacciones
   • Selección de amigos
   • División equitativa/proporcional
   • Envío WhatsApp

4. 💰 DETALLE APARTADO
   • Progreso del grupo
   • Lista de participantes
   • Sistema de votaciones
   • Historial de movimientos

5. 🏦 CRÉDITO COMPARTIDO
   • Simulador en tiempo real
   • Slider de monto
   • Sistema de firmas
   • Cálculo automático

6. 📜 HISTORIAL
   • Todas las transacciones
   • Búsqueda
   • Filtros por tipo
   • Estados visuales

7. ➕ CREAR GRUPO
   • Formulario completo
   • Selector de colores
   • Agregar participantes
   • Preview en vivo

┌─────────────────────────────────────────────────────────────────────┐
│  ❓ PROBLEMAS COMUNES                                                │
└─────────────────────────────────────────────────────────────────────┘

❌ "npm no se reconoce"
   → Instalar Node.js: https://nodejs.org/

❌ "Cannot find module"
   → Ejecutar: npm install

❌ Puerto 3000 ocupado
   → Vite te ofrecerá otro puerto automáticamente

❌ No se ve bien en el navegador
   → Activar vista móvil (F12 → Ctrl+Shift+M)
