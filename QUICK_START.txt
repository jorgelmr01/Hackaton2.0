  FINANZAS OPENFRIENDS - QUICK START GUIDE
═══════════════════════════════════════════════════════════════

📋 PASO 1: INSTALAR NODE.JS
─────────────────────────────────────────────────────────────
1. Ir a: https://nodejs.org/
2. Descargar versión LTS (recomendada)
3. Ejecutar instalador
4. Reiniciar terminal/PowerShell

📦 PASO 2: INSTALAR DEPENDENCIAS
─────────────────────────────────────────────────────────────
Abrir terminal en la carpeta del proyecto y ejecutar:

   npm install

⏳ Esto tomará 1-2 minutos

🚀 PASO 3: INICIAR APLICACIÓN
─────────────────────────────────────────────────────────────
Ejecutar:

   npm run dev

✅ La app se abrirá automáticamente en http://localhost:3000

📱 PASO 4: VISTA MÓVIL
─────────────────────────────────────────────────────────────
1. Presionar F12 (DevTools)
2. Presionar Ctrl+Shift+M (Vista móvil)
3. Seleccionar iPhone o Pixel
4. ¡Listo! Explora la app

  PROBLEMAS COMUNES
═══════════════════════════════════════════════════════════════

   → Instalar Node.js primero

❌ "Cannot find module"
   → Ejecutar: npm install
